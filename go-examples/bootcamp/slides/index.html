Why Go?

Why Go?

Reaction to “speed or ease of use; pick one.”
  • complexity, weight, and noise (C++, Java)
  • nothing statically checked (JavaScript, Python)

Go is statically typed and compiled, like C++ or Java, but in many ways feels as lightweight and dynamic as JavaScript or Python.

Goals for Go

Compile fast.

Feel lightweight on the page.

Feel flexible, adaptable.

Allow interesting, productive tools.

Allow enough control to write efficient programs.

Methods

Methods can be associated with any named type.

but only in the package that declares the type

type Point struct {
    X, Y float64
}

func (p *Point) Abs() float64 {
    return math.Sqrt(p.X*p.X + p.Y*p.Y)
}

p := Point{3, 4}
h := p.Abs()
// h == 5

Methods

Methods can be associated with any named type.

type MyFloat float64

func (f MyFloat) Abs() float64 {
    if f < 0 {
        return -f
    }
    return f
}

Interfaces

Interface types declare a set of methods.

type Abser interface {
    Abs() float64
}

Interfaces

An interface value can hold any value that implements the methods.

var (
    a Abser
    f MyFloat
    p Point
)

a = f   // ok
a = &p  // ok
a = p   // ERROR: Abs method receiver is *Point, not Point

a.Abs() // method invocation

Interfaces

Types implement interfaces by implementing methods

  • No explicit declaration of intent.
  • Decouples packages - neither depends on the other.
  • Allows more precise interfaces 
type Reader interface {
    Read(b []byte) (n int, err os.Error)
}

type Writer interface {
    Write(b []byte) (n int, err os.Error)
}

type ReadWriter interface {
    Reader
    Writer
}

Errors

An error is anything that can describe itself: 
package os

type Error interface {
	String() string
}

Web servers

Package http serves HTTP requests using any value that implements http.Handler:

package http

type Handler interface {
    ServeHTTP(conn ResponseWriter, req *Request)
}

Web servers

As a convenience, package has global path-based multiplexor:

func Handle(path string, h Handler)
func HandleFunc(path string, f func(http.ResponseWriter, *http.Request))

Pass nil to Serve or ListenAndServe to use the global multiplexor.
